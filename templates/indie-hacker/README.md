# Indie Hacker Team

Full engineering team for solo developers and micro-SaaS builders.

## Who is this for?

Solo founders and indie developers who need an entire engineering team — architect, developer, code reviewer, and RFC author — without the headcount.

## Agent Team

| Role | Model | Responsibility |
|------|-------|---------------|
| Tech Lead | claude-sonnet-4-6 | Architecture decisions, push back on complexity, fast decision-making |
| Full-Stack Engineer | claude-sonnet-4-6 | Write clean, working code with boring technology and basic error handling |
| QA Reviewer | claude-haiku-4-5 | Find bugs, edge cases, security issues, and future technical debt |
| RFC Author | claude-sonnet-4-6 | Write concise architectural decision documents (activated by design keywords) |

## Example Goal

```
Build user authentication with magic links for SaaSKit
```

## Estimated Cost

~$0.12 per run (recommended minimum 2 runs)

## Tips for Best Results

- Start with architecture/design goals to get an RFC before jumping to implementation
- The tech lead actively pushes back on over-engineering — trust it
- Use specific feature descriptions rather than vague goals
- Run QA after implementation runs to catch edge cases
